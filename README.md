## Requirements

PHP >= 7.4

Node >= 10.17.0 

Npm >= 6.14.2

Apache or nginx for local hosting 

## Installation

1. Copy and rename .env.example to .env
2. Run composer install
3. Run npm install
4. Generate encryption key using php artisan key:generate
5. Done!
